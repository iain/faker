= Faker

A port of Perl's Data::Faker library that generates fake data.

== Usage
* Faker::Name.name => "Christophe Bartell"

* Faker::Internet.email => "kirsten.greenholt@corkeryfisher.info"

== Original repository

This is a just simple checkout.

The original SVN repository can be found at:
  svn://rubyforge.org/var/svn/faker/trunk

== This Fork

This is a dutch version, creating dutch names etc.

== Install

  gem install iain-faker
